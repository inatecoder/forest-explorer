An interactive, 2D animated educational website exploring **Chapter 15: Forests: Our Lifeline**.

## Features
- **Parallax Hero Section**: Smooth entrance animation.
- **Interactive Layers**: Explore Emergent, Canopy, Understorey, and Forest Floor.
- **Photosynthesis Game**: Plant trees to balance CO2 and O2 levels.
- **Food Chain Quiz**: Drag-and-drop logic puzzle.

## Setup
1. Clone the repo.
2. Open `index.html`.

## Deployment
This site is ready for GitHub Pages.
1. Upload to GitHub.
2. Go to Settings > Pages > Source: master branch.
